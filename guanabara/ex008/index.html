Este comando abre uma janela de opções para serem aplicadas no texto selecionado.

Tutorial: selecione o texto que deseja deixar entre Tags. Aperte Ctrl + Shift + P e digite "Wrap with Abbriviation", dê Enter e digite a tag desejada
